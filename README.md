# 6502emu
6502 Emulator

Prerequisites:
VSCode / VSCodium (open source)

*Linux Open Sources extensions*

+ cmake
+ c++ extension pack
+ Cmake
+ CMake Tools
+ CodeLLDB
+ NativeDebug

VSCode / Microsoft
*Windows extensions*
    
+ CMake
+ CMake Tools
+ C++ Extensions Mocrosoft


*Tests*
After build run 
```
./run_tests.sh
```
